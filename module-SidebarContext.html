Module: SidebarContext

Why is this needed?

  • Child components (SidebarItem, SidebarAccordionItem) need to know the sidebar's open/collapsed state to determine whether to show only icons or full labels
  • Passing props through multiple nesting levels causes prop drilling
  • Context allows any depth component to access state via useSidebarContext()
