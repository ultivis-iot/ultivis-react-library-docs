@ultivis-iot/react. Namespace

layouts

Description:

  • Layout components for application structure.

    This module provides the main layout system including:

    • Layout: Main application layout component with header, sidebar, and content area
    • Header: Application header with navigation, app switcher, and user menu
    • Sidebar: Collapsible sidebar navigation with menu items and accordions
    • MenuBar: Action bar and tab navigation for page-level controls

    All layout components are responsive and support dark mode.
